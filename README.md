# 📊 GroupBy, Bar, Line & Pie Chart Analysis

A Python data visualization project demonstrating how to perform **data aggregation using GroupBy** and create insightful **Bar, Line, and Pie Charts** with Pandas and Matplotlib. This project showcases essential data analysis and visualization techniques commonly used by Data Analysts.

---

## 📌 Project Overview

This project focuses on transforming raw data into meaningful visual insights using Python. It covers data grouping, aggregation, and visualization to help understand trends, comparisons, and distributions effectively.

---

## 🎯 Objectives

- Perform data aggregation using `groupby()`
- Analyze categorical and numerical data
- Create Bar Charts for comparisons
- Create Line Charts for trend analysis
- Create Pie Charts for distribution analysis
- Extract meaningful business insights
